# Linux Command 
Command: `cd` `touch` `mkdir` `cp` `mv` `rm` `pwd` `ls` `cat`

Flag: Flag start with `-` | `r` for recursive | `a` for all | `f` for force

Identifiers: `*` for everything | `.` for current directory | `..` for  parent directory

To Change directory
```
cd <folderName>
```

To make a file
```
touch <fileName> 
```

To make a directory
```
mkdir  <folderName>
```

To copy
```
cp <copyPath> <pastePath>
```

To move
```
mv <copyPath> <pastePath>
```

To see what inside a directory
```
ls
```

To remove file
```
rm <fileName / orfolderName>
```

To know the present working directory
```
pwd
```

To quick read file
```
cat <filename> 
```
